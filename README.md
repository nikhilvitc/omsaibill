# OM SAI FURNITURE - Invoice Generator

A web-based invoice generation system for OM SAI FURNITURE AND ELECTRONICS.

## Features

- **Professional Invoice Generation**: Create detailed invoices with itemized billing
- **Customer Information Management**: Store and manage buyer details
- **Automatic Calculations**: Real-time calculation of totals, taxes, and discounts
- **PDF Export**: Generate professional PDF invoices for printing and sharing
- **Local Storage**: Automatic invoice numbering with browser storage
- **Responsive Design**: Works on desktop and mobile devices

## Getting Started

1. Open `index.html` in your web browser
2. Fill in the customer details
3. Add items to the invoice using the "Add Item Row" button
4. The system will automatically calculate totals and taxes
5. Generate a PDF using the "Generate PDF" button
